# Типы данных

## Расширяемые типы данных
Данные типы данных используются в `ConfigLoader` для расширения 
данных по-умолчанию данными из директории приложения

## `EList`
Расширяемый список

## `EDict`
Расширяемый словарь

## `DirectoryType`
Тип файлов, обозначающий директорию. Используется в `AssetsManager`
